This repo is developed in C# and Entity Framework Code First using N-Tier Architecture. 
There are 3 types of user accounts. They include:

1. Admin (roleID = 1)
2. Teacher (roleID = 2)
3. Student

**FUNCTIONS OF ACCOUNTS**

**-- Admin**
* Controlling teacher and student login and logout times
* Create, Delete and Manage students
* Create, Delete and Manage staffs
* Create, Delete and Manage subjects
* Create, Delete and Manage classes, also subject can be added for it
* Controlling students' attendance
* Create, Delete and Manage roles
* Create, Delete and Manage exam types
* Controlling the status of the examination of students
* Sending SMS to students

**-- Teacher**
* Controlling own students' attendance
* Grading exams
* Sending SMS to students

**-- Student**
* Checking their own attendance
* Checking their own exam results
* Reading incoming SMS
